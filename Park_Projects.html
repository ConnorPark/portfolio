App Inventor Project

September 2018

This project was longer than the Scratch Project, using similar block software, but for Phone Apps instead of computer programs. This project was done with Abhishek Sarepaka and we spent most of this project together brainstorming and coding together. The game is a simple high score game, the goal being to get as many baskets as possible. If you touch one of the rotten bananas, you would lose a point.

We wanted this game to have a similar feel to a game that focuses on getting a good high score within a certain time period. A feature that I am especially proud of is the WebDB, so people can compare high scores with other people across the world. The game was common, but the storyline is unique, similar to the Scratch project.

During the making of the game, there was a big problem that both me and Abhi faced. After coding the high score, the game wouldn't run! It was stuck and we would get an error code! It turns out that we used mathematical statements instead of normal statements. It was very stressful but I learned a valuable lesson about comupter science. I can use this in the future to make better projects.
